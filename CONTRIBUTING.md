# Contributing to the repo

* Pull Requests (PRs) can be created but there is currently no guarantee that they will be reviewed or integrated in a timely manner.
* If you're not seeing any traction on your PR, please reach out on the [QNX Discord server](https://discord.com/invite/Jj4EkkrFTT) or [qnxcommunityengagement@qnx.com](mailto:qnxcommunityengagement@qnx.com).
